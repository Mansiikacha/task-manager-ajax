# Task Manager - PHP AJAX Application

## Features
- ✅ User Registration & Login
- ✅ Add/Edit/Delete Tasks  
- ✅ AJAX - No Page Reload
- ✅ Task Status Toggle (Pending/Completed)
- ✅ Statistics Dashboard
- ✅ Responsive Design (Tailwind CSS)

## Technologies
- PHP (Core)
- MySQL (Database)
- AJAX (Fetch API)
- Tailwind CSS
- Git & GitHub

## Security
- Password Hashing
- SQL Injection Prevention (Prepared Statements)
- XSS Protection
- Session Authentication

## Installation
1. Clone repository
2. Import database
3. Update `config/database.php`
4. Run on XAMPP/WAMP
